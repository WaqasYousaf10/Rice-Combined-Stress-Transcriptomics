## Project Overview

This repository contains all bioinformatics and data analysis scripts for my PhD research on combined submergence and iron deficiency tolerance in rice.

### Research Question
Does the SUB1A-1 ERFVII transcription factor enhance tolerance to combined submergence and iron deficiency by regulating Fe uptake genes?

### Experimental Design

| Factor | Details |
|--------|---------|
| Genotypes | IR64, IR64-Sub1, FR13A, Swarna, Swarna-Sub1, rice landraces |
| Treatments | Control, Iron deficiency, Submergence, Combined (Sub+Fe) |
| Time points | 0, 3, 7, 14 days stress; 7 days recovery |
| Replicates | 3 biological replicates per condition |
